
# Exploring Earth's Pinnacles - A Journey to the World's Highest Mountains

## Project Description
This project showcases an article titled "Exploring Earth's Pinnacles - A Journey to the World's Highest Mountains," accompanied by a responsive website. The article covers four notable mountain ranges: Himalayas, Karakoram, Hindu Kush, and Kongur Shan. The website features a navigation menu dynamically populated with JavaScript, smooth scrolling between sections, a "Go to Top" button, and a visually appealing design.

## Usage
To explore the content, open the index.html file in a web browser. The website provides information on each mountain range, their unique characteristics, and the cultural significance they hold. The navigation menu allows users to quickly jump to specific sections, enhancing the overall user experience.

## Dependencies
The project relies on the following files:

index.html: The main HTML file containing the structure of the webpage.
css/style.css: The stylesheet defining the visual presentation and responsiveness of the website.
js/app.js: The JavaScript file populating the navigation menu and handling smooth scrolling and other interactive features.
